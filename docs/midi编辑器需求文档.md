# 项目简介

一个基于网页的 MIDI 编辑器，我希望这个 midi 编辑器能够对用户输入的现成 midi 进行修改，或是直接在这个 midi 编辑器里制作生成一个新的 midi,让用户可以简单编辑音符、试听，并导出 MIDI 文件或 mp3 文件。

# 功能清单

- [x] 在网页上显示全钢琴卷帘（Piano Roll），这个钢琴卷帘可以放大或缩小或是水平移动，以便用户在特定音域内制作，可以添加、移动、删除音符。
- [x] 能实时试听编辑的内容，还可以指定在哪一处开始试听。
- [x] 可以用鼠标拖动音符块的边缘部分来修改音符的时值，也可以直接输入这个音符持续多少秒来进行绝对修改，也可以通过选择这个音符是四分音符或是八分音符等音符模式来进行相对修改。
- [x] 可以直接修改 midi 的 bpm
- [x] 具有撤销、保存、另存为功能
- [x] 编辑完成后可以保存下载 MIDI 文件或 mp3 文件。
- [x] 支持多轨编辑，可以分轨试听，可以自行决定各音轨的音量大小
- [x] 可导入已有 MIDI 文件进行编辑。
- [ ] 可导入音源(可选)
- [ ] 具有简单的混音功能，如调节各音轨的响度等(可选)
- [ ] 编辑器的功能区分栏清晰美观
- [ ] 左键点击音符块（使音符处于选中状态），能发出该音符块对应的音，音符块每移动到网格的一个位置时，也要发出其位置对应的音，每个音符块都可以显示其对应的音
- [ ] 用右键或退格键或delete键删除音符块
- [ ] 可以通过电脑的 ↑↓←→ 键来选择音符块，每选择到一个音符块，发出该音符块的音，↑↓ 键负责选择同一时刻的音
- [ ] 能够听和弦（同一时刻的音）
- [ ] 支持三连音、附点音符等
- [ ] 时间轴最小刻度为 32 分音符，允许用户切换主界面网格分辨率（如 1/4、1/8、1/16、1/32）
- [ ] 支持对音符块的批量编辑(如把指定的音符块批量后移一拍，把指定音符块集体升 2key)
- [ ] 钢琴卷帘有水平和竖直两个形态，用户可自行选择
- [ ] 支持启用键盘来放置音符块
- [ ] 支持用键盘演奏并把演奏部分制作成 midi
- [ ] 支持设置某个乐段的bpm
- [x] 嵌入Agent（模型可用DeepSeek），让Agent辅助用户进行编辑。具有Agent对话聊天窗口，且能记录之前所有的对话，用户可自行增加、删除对话。Agent具有一定的乐理基本知识，可以根据用户的需求，自动生成MIDI文件。Agent还可以根据用户的需求，自动生成midi音符，并让用户进行编辑。

## 前端钢琴样式设计


# 非功能性需求

- 界面简洁、易用，新手不用看说明也能操作。
- 兼容主流 PC 浏览器（Chrome/Edge/Firefox）。
- 无需注册登录即可试用

# 技术偏好

- 前端建议用 JS/Typescript/Vue，支持 Tone.js 等 MIDI 库。
- 后端不确定。

# 输入与输出

- 输入：用户鼠标操作或上传.midi 文件。
- 输出：可下载的标准 MIDI 文件。

# 哼唱识别转MIDI功能（重要！！）

- 支持用户通过麦克风录制哼唱（单音或简单旋律）

- 使用AI模型（如CREPE、YAMNet + 音高追踪）识别音高与节奏

- 将识别结果转换为MIDI音符，并自动创建新音轨

- 用户可预览识别结果，并手动修正识别错误的音符

- 支持将识别后的MIDI直接载入编辑器进行进一步编辑

# 其他具体细节

- 钢琴卷帘的可视音域范围为 C1~C8，需要支持“吸附到网格”功能，便于精准放置音符
- 试听时播放全部轨道，但可调节各音轨的音量，最多支持 16 音轨，支持轨道重命名、颜色区分

# 示例场景

1. 用户打开网页，看到空白的钢琴卷帘。
2. 用户点击/拖拽添加音符，按下播放按钮试听。
3. 用户点击“导出 MIDI”或“导出为 mp3 文件”，下载编辑好的文件。
